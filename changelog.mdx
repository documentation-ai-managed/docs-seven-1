## Recent Releases

Stay informed about Seven's evolution. You receive new features, performance enhancements, and critical bug fixes through regular releases. Check the details below for what's new.

<Update label="2024-10-15" description="v1.3.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaboration for shared documentation spaces, allowing multiple users to edit simultaneously.
- Introduced version history with diff views, so you can compare changes across document revisions.

## Improvements

- Enhanced search functionality with fuzzy matching and full-text indexing for faster results.
- Optimized rendering performance, reducing load times by up to 40% for large documentation sites.

## Bug Fixes

- Fixed markdown parsing issues with nested lists and code blocks.
- Resolved authentication token refresh errors during long editing sessions.

</Update>

<Update label="2024-09-20" description="v1.2.0" tags={["feature", "bugfix"]}>

## New Features

- Integrated custom themes using your brand color `#3B82F6` and CSS variables for personalized styling.
- Added export options to PDF and HTML, enabling easy offline sharing of your docs.

## Bug Fixes

- Corrected sidebar navigation glitches on mobile devices.
- Patched security vulnerability in public space sharing links.

</Update>
